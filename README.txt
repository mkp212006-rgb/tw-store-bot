BOT TELEGRAM - TW STORE

1. Crie seu bot no Telegram falando com @BotFather.
2. Copie o token gerado.
3. Instale Python no celular/PC/servidor.
4. Dentro da pasta do projeto, instale:
   pip install -r requirements.txt

5. Configure o token:
   Windows PowerShell:
   $env:TELEGRAM_BOT_TOKEN="SEU_TOKEN_AQUI"

   Linux/Termux:
   export TELEGRAM_BOT_TOKEN="SEU_TOKEN_AQUI"

6. Inicie o bot:
   python bot.py

O bot já vem com os catálogos:
- Instagram Seguidores
- Instagram Curtidas
- Instagram Visualizações

Para alterar valores, edite as listas SEGUIDORES, CURTIDAS e VISUALIZACOES dentro do bot.py.
